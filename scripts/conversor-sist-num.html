Conversor de sistema numérico

Script que converte números entre diferentes sistemas numéricos, como Binário, Decimal, Hexadecimal e Octal.

Código

      
        //exercício: faça uma aplicação que receba um número decimal digitado pelo usuário e apresente o valor nos sistemas binário, octal e hexadecimal

        // requerimento de inicialização do módulo
        const prompt = require("prompt-sync")();
        
        //permitir o input/entrada de dados do usuário
        let decimalNumber = parseFloat(
          prompt("Qual número você deseja converter entre sistemas numéricos? "),
        );
        
        //indicar operação de conversão para cada sistema numérico
        let binary = decimalNumber.toString(2); // sistema numérico binário (base 2)
        let octal = decimalNumber.toString(8); // sistema numérico octal (base 8)
        let hexadecimal = decimalNumber.toString(16); // sistema numérico hexadecimal (base 16)
        
        console.log(
          `O número decimal ${decimalNumber} em número binário corresponde a ${binary}; \n já em número octal corresponde a ${octal}; \n enquanto em número hexadecimal corresponde a ${hexadecimal}`,
          );
