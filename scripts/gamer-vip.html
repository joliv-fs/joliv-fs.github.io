Rank Gamer VIP

Script que confere o status de um jogador com base em sua pontuação.

Código

      
        /*
        exercício: o usuário de uma página de games deve inserir sua pontuação: se for acima de 1.000, o programa exibirá “Usuário VIP”; caso contrário, mostrará “Aumente sua pontuação para se tornar um usuário VIP”.
        */

        //requires: inicialização de módulos
        const prompt = require("prompt-sync")();

        //vars: declarar variáveis necessárias
        let pontuacaoUsuario = 1.0; //colocando assim ele identifica tanto 1.001 quanto 1001

        //user input: entrada de dados por parte do usuário
        pontuacaoUsuario = parseFloat(prompt("Qual foi a sua pontuação? "));

        //lógica de operação do script
        let statusUsuario =
          pontuacaoUsuario >= 1.0
            ? "Parabéns! Você é um Usuário VIP!"
            : "Aumente sua pontuação para se tornar um usuário VIP.";
        console.log(statusUsuario);
