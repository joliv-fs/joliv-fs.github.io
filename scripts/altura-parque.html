Conferir altura

Script que confere se o visitante do parque de diversões pode entrar , ou não, no brinquedo.

Código

      
        /*
        exercício: Um parque solicitou um script que, ao informar a altura da criança, permita o acesso se for igual ou maior que 1,20 m; caso contrário, exiba “Acesso apenas para crianças com mais de um metro e vinte”.
        */

        //requires: inicialização de módulos
        const prompt = require('prompt-sync')();

        //vars: declarar variáveis necessárias
        let alturaCriança = 1.20;

        //user input: entrada de dados por parte do usuário
        alturaCriança = parseFloat(prompt("Por motivos de segurança, informe a altura da criança: "));

        //lógica de operação do script
        let statusAcesso = (alturaCriança >= 1.20) ? "Acesso negado! O brinquedo é permitido somente para crianças com mais de um metro e vinte de altura." : "Acesso liberado!"; 
        console.log(statusAcesso);
