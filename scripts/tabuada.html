Tabuada

Script que gera a tabuada de um número informado pelo usuário.

Código

      
        //exercício: apresente a tabuada de um dado numero x, até um multiplicador y (por exemplo, x =5 e y = 7 então deve apresentar resultdos de 5*1 até 5*7)

        //while de cara e for pedindo dado user

        //requires: inicialização de módulo
        const prompt = require("prompt-sync")();

        //var: declaração de variáveis ; nome descritivo para você se lembrar ao que ela se refere
        let numeroMultiplicando = null; 
        let numeroMultiplicador = null;

        //input: entrada do usuário, coleta de dados
        numeroMultiplicando = parseInt(prompt("Informe o número que você deseja multiplicar: ")); 
        numeroMultiplicador = parseInt(prompt("Informe até que número você deseja multiplicar: ")); 
        /*aqui pode ser parseInt porque consideramos que o usuário vai entrar números inteiros
        no caso de numerais com dois digitos, o parseInt dá erro porque ele interpreta como dois numerais
        */

        //lógica de operação do script: 
        for (let i = 1; i <= numeroMultiplicador; i++) {
          console.log( numeroMultiplicando * i);
          }
