Calculadora de comissão sobre vendas

Script que calcula a comissão dos vendedores com base no valor total de vendas de um estabelecimento comercial.

Código

      
        //exercício: desenvolva um programa para calcular comissão com as seguintes condições: 4% se vendas mensais forem ≥ R$ 100.000,00; 2% se forem menores.

        //requerimento de inicialização de módulo
        const prompt = require("prompt-sync")();

        //entrada de dados por parte do usuário
        let comissao = 0;
        const vendasMensal = parseFloat(prompt("Qual foi o total de vendas do mês? "));

        //lógica de operação do script
        if (vendasMensal >= 100000) {
          console.log("Aplicar 4% de comissão");
          comissao = vendasMensal * 0.04;
          console.log("Sua comissão será de", comissao);
        } else {
          console.log("Aplicar 2% de comissão");
          comissao = vendasMensal * 0.02;
          console.log("Sua comissão será de", comissao);
        }

        //aqui poderia ser feito tratamento de erro, colocar 'loop'
