Níveis de saúde (colesterol e glicose)

Script que classifica o nível de colesterol a partir de dados informados pelo usuário, seguindo regras pré-definidas.

Código

      
        //exercício: faça um programa que, ao informar o colesterol, exiba “Saúde Ok” se for menor que 180, ou “Vamos procurar uma ajuda médica?” caso contrário.
        
        //requerimento de inicialização de módulo
        const prompt = require('prompt-sync')();
        
        //entrada de dados por parte do usuário
        const nivelColesterol = parseFloat(prompt('Qual é o seu nível de colesterol atual? '));
        
        //lógica de operação do script
        if (nivelColesterol < 180)
          console.log('Saúde OK.');
        else (nivelColesterol > 180)
          console.log('Vamos procurar uma ajuda médica?');
      
    

      
        //exercício: repita o exercício anterior, agora com glicose, e usando intervalo para classificar
        
        //requerimento de inicialização de módulo
        const prompt = require('prompt-sync')();
        
        //entrada de dados por parte do usuário
        const nivelGlicose = parseFloat(prompt("Qual é o seu nível de glicose atual? "));
        
        //lógica de operação do script
        //fora do intervalo
        if (nivelGlicose < 70 || nivelGlicose > 180){ 
          console.log("Vamos procurar uma ajuda médica?");
        //dentro do intervalo
        } else if (nivelGlicose > 70 && nivelGlicose < 180){ 
          console.log("Parabéns! Sua glicose está no intervalo ideal.");
        }
